# Master Quiz App

A comprehensive quiz application with timer and multiple questions, built with HTML, CSS, and JavaScript.

## Features

- 20 multiple-choice questions on various topics
- 30-second timer per question
- Real-time score tracking
- Visual feedback for correct/incorrect answers
- Total time taken display
- Responsive design

## How to Play

1. Open `master_quiz.html` in your web browser.
2. Answer each question by clicking on an option before the timer runs out.
3. See immediate feedback on your choice.
4. Click "Next Question" to proceed.
5. At the end, view your final score and total time.
6. Click "Restart Quiz" to play again.

## Topics Covered

- Geography
- Science
- History
- Literature
- General Knowledge

## Files

- `master_quiz.html`: Main structure
- `master_quiz.css`: Styling and responsive design
- `master_quiz.js`: Quiz logic with timer

## Note

Questions are randomized in order each time you restart.
