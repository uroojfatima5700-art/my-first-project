# Parrot Puzzle Game

A fun memory matching game with colorful parrot images, built with HTML, CSS, and JavaScript.

## Features

- 16 cards with colorful parrot placeholder images
- Flip cards to find matching pairs
- Move counter and timer
- Win message with stats
- Reset and play again options

## How to Play

1. Open `parrot_puzzle.html` in your web browser.
2. Click on cards to flip them and reveal the parrot image.
3. Try to find matching pairs of parrot images (same color).
4. If two cards match, they stay flipped; otherwise, they flip back after a second.
5. Match all 8 pairs to win!
6. Use the reset button to start a new game anytime.

## Files

- `parrot_puzzle.html`: Game structure
- `parrot_puzzle.css`: Styling with animations
- `parrot_puzzle.js`: Game logic

## Note

The game uses placeholder images with parrot emojis for a fun, colorful theme.
