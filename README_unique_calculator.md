# Unique Calculator

A unique calculator app built with HTML, CSS, and JavaScript, featuring:

- Light/Dark theme toggle
- Calculation history (last 5 entries)
- Parentheses for complex expressions
- Backspace button
- Retro-style design with hover animations

## Usage

1. Open `unique_calculator.html` in your web browser.
2. Use the buttons to input numbers and operators.
3. Click "=" to calculate.
4. Use "C" to clear, "âŒ«" to backspace.
5. Toggle theme with the moon/sun button.
6. View history below the display.
