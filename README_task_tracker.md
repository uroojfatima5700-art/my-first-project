# Task Tracker App

A simple task tracker (to-do list) app built with HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Mark tasks as completed
- Edit existing tasks
- Delete tasks
- Filter tasks by status (All, Pending, Completed)
- Persistent storage using localStorage

## Usage

1. Open `task_tracker.html` in your web browser.
2. Enter a task in the input field and click "Add Task" or press Enter.
3. Check the checkbox to mark a task as completed.
4. Click "Edit" to modify a task.
5. Click "Delete" to remove a task.
6. Use the filter buttons to view tasks by status.

## Files

- `task_tracker.html`: Main HTML structure
- `task_tracker.css`: Styling
- `task_tracker.js`: Functionality with localStorage

## Note

Tasks are saved in your browser's localStorage, so they persist between sessions.
